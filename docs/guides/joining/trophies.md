# Trophies

Good news: every trophy in the game still pops. Two players have already grabbed the Platinum on Camp Crystal Lake. **Congratulations to them!** Bad news: PSNProfiles doesn't like custom servers. More on that below.

## What works

- **All 41 trophies**, Platinum included
- **Cross-region autopop** works the same as on the official servers. Finish a trophy's requirements on one region's stack and it pops on the others
- **Group trophies** (full-lobby kills, 8-player matches, etc.) can be boosted in `#boosting` on [Discord](https://discord.gg/dn4pacGHe3)

## Anything broken?

Not that we've seen. If a trophy doesn't pop when it should, drop a note in `#bug-reports` on [Discord](https://discord.gg/dn4pacGHe3) with what you did. You might earn the **Bug Killer** role for the trouble.

## Boosting sessions

Some trophies need a full lobby or coordinated play. Head to `#boosting` on [Discord](https://discord.gg/dn4pacGHe3) and organize with whoever's around.

!!! tip "The 7-counselor trophies can be done with only 3 people"

    Check the pinned messages in `#boosting` on [Discord](https://discord.gg/dn4pacGHe3) for the method. Saves you from needing a full lobby.

## PSNProfiles

!!! warning "PSNP will flag you"

    PSNProfiles considers connecting to a custom server a flaggable offense. Any trophy you earn on Camp Crystal Lake will get you removed from the PSNP leaderboards for this game.

    For a custom server to be approved by PSNP it has to be fully open source, among other things. Our main server is **not** open source (and won't be -- see [About](../community/about.md) for why). So we fail that check.

    That said, the standalone `ccl-server` **will be** open source. Once that's out in July 2026 and stable, we plan to talk to the PSNP moderators about getting that path approved.

!!! tip "Your PSN account is safe"

    Sony doesn't care about custom servers. Your PSN account won't be banned or suspended. This is purely a PSNP leaderboard issue.
